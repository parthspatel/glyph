Create the ModuleCard component with the following structure:

```tsx
interface ModuleCardProps {
  title: string;
  icon: React.ReactNode;
  status: 'configured' | 'unconfigured' | 'warning';
  preview: React.ReactNode;
  emptyText?: string;
  emptyAction?: string;
  onClick: () => void;
  isLoading?: boolean;
}
```

**Visual states:**
- `configured`: Normal card bg, solid border, preview content visible
- `unconfigured`: `bg-muted/30` background, dashed border (`border-dashed`), shows emptyText + emptyAction button
- `warning`: `border-warning` left accent (4px), warning icon next to title
- Hover: `hover:shadow-md hover:-translate-y-0.5 transition-all duration-150`

**Structure:**
- Use shadcn Card as base with `p-4` padding (not default p-6)
- Header: icon (size-5) + title (text-sm font-medium) + optional warning icon
- Content: preview slot OR empty state guidance
- Empty state: centered icon + muted text + outline action button

**Styling (from research):**
- Card: `rounded-lg border bg-card cursor-pointer`
- Interactive: Add `role="button"` and `tabIndex={0}` for accessibility
- Use design tokens: `text-foreground`, `text-muted-foreground`, `bg-muted/30`
