### Key Features

**StepConfigPanel:**
- 75vw width slide-out from right (per CONTEXT.md "3/4 width")
- Opens when node selected, closes on outside click or Escape
- Uses shadcn Sheet component with custom width
- Scrollable content area

**PanelHeader:**
- Step type icon with color coding
- Inline editable step name
- Delete button with confirmation dialog
- Close button

**StepConfigContent:**
- Routes to appropriate form based on step type
- Placeholder forms showing expected sections
- Actual forms to be implemented in Plan 06

**useStepConfig Hook:**
- `config` — Current step configuration
- `updateConfig()` — Update config in store
- `updateSettings()` — Update settings helper
- `errors` — Validation errors
- `hasErrors` — Boolean flag
- `isDirty` — Unsaved changes flag
