# Plan 09.1-07 Summary: Data Source Wizard

## Delivered

### Wizard Steps (apps/web/src/components/workflow/DataSource/WizardSteps/)

**SourceTypeStep.tsx**
- Card-based selection grid for data source types
- Types: File Upload, S3, GCS, Azure Blob, API, Database
- Visual icons and descriptions for each type

**ConnectionStep.tsx**
- Dynamic form based on selected source type
- S3: bucket, region, prefix, IAM role or credentials
- GCS: bucket, prefix, service account JSON
- Azure Blob: container, connection string
- API: endpoint, auth type (none/bearer/basic/api_key)
- Database: host, port, database, username, password
- File Upload: drag-drop zone with file type filter
- Test Connection button with status indicator

**SchemaStep.tsx**
- Input schema editor (JSON Schema draft-07)
- Output schema editor
- "Infer from File" button for uploaded files
- Uses SchemaEditor Monaco component

**ConfirmStep.tsx**
- Summary of all configuration
- Data source name input
- Validation checklist with pass/fail indicators
- Blocks creation if validation fails

### Main Components

**DataSourceWizard.tsx**
- 4-step progress indicator
- Back/Next navigation
- State management for all wizard data
- Form validation per step
- Submit to create data source

**SchemaEditor.tsx**
- Monaco editor for JSON Schema
- Format, Copy, Reset toolbar
- Default schema template

**DataPreview.tsx**
- Single row display with JSON highlighting
- First/Prev/Next/Last navigation
- "Jump to row" input
- Schema validation with error display
- Row counter (e.g., "Row 3 of 150")

**DataSourceForm.tsx**
- Direct edit mode for existing data sources
- Collapsible sections: Connection, Input Schema, Output Schema
- Save/Delete buttons
- Test Connection
